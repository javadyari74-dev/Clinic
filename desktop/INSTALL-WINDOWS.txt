ساخت فایل نصب ویندوز از این بسته (نسخه از پیش ساخته‌شده)
=========================================================

این بسته شامل خروجی‌های آماده‌ی نرم‌افزار است (پوشه‌ی dist). شما فقط کافی است
روی ویندوز فایل نصب‌کننده (.exe) را بسازید — نیازی به نصب pnpm یا ساخت مجدد
کدها نیست.

پیش‌نیاز:
  - Node.js نسخه ۱۸ یا بالاتر  (از https://nodejs.org نصب کنید)

مراحل (در PowerShell یا CMD):

  1) این فایل zip را باز کنید (Extract).

  2) وارد پوشه desktop شوید:
       cd desktop

  3) وابستگی‌ها را نصب کنید (این کار باینری ویندوزی دیتابیس را نصب می‌کند):
       npm install

  4) فایل نصب‌کننده ویندوز را بسازید:
       npm run dist:prebuilt

  >>> توجه مهم: حتماً از  dist:prebuilt  استفاده کنید (نه dist).
      دستور dist تلاش می‌کند کدها را دوباره بسازد که روی ویندوز کار نمی‌کند.
      dist:prebuilt مستقیم از خروجی‌های آماده‌ی داخل همین بسته استفاده می‌کند.

پس از پایان، فایل نصب‌کننده اینجا ساخته می‌شود:

  desktop/release/DoctorYariClinic-Setup-1.0.0.exe

آن را اجرا کنید تا نرم‌افزار نصب شود.

  npm install
  npm start
